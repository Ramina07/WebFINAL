## Oishiro – Japanese Cuisine Website

### General Overview

**Oishiro** is an interactive Japanese restaurant website created to demonstrate front-end development skills and the integration of modern web technologies.
The project includes responsive design, authentication, navigation between pages, interactive UI elements, and a visually appealing pink-themed interface inspired by Japanese culture.

---

### ⚙️ Technologies Used

* **HTML5** – for page structure and content organization.
* **CSS3** – for design, color schemes, typography, spacing, and responsiveness.
* **JavaScript** – for interactivity, validation, and logic handling (form validation, dynamic content, and popup control).
* **jQuery** – to simplify DOM manipulation, event handling, and animations.
* **Bootstrap 5** – for layout grids, responsive components, and overall UI consistency.

---

### Website Structure

1. **Home Page (index.html)** – Welcome screen with authentication (Sign Up / Sign In).
2. **Menu Page (menu.html)** – Interactive food menu with images, descriptions, and order buttons.
3. **About Page (indexr.html)** – Information about the restaurant’s story and Japanese cuisine philosophy.
4. **Contact Page (contact.html)** – Contact form and location details for customer feedback.

---

### Implemented Topics and Features

#### Navigation

* A responsive **navigation bar** connects all main pages.
* The active page is highlighted for better UX.
* After login, a **profile icon with the user’s name** appears in the header.

#### Authentication

* **Sign Up / Sign In popup** appears on the first site visit.
* JavaScript-based form validation (Name and Email are required).
* After login, the user is redirected to the homepage, and their name is displayed in the profile section.

#### Bootstrap & Layout

* **Bootstrap grid system** used for flexible page layout.
* Components like `container`, `row`, `col`, `card`, and `btn` were applied.
* Ensures a clean, consistent, and mobile-friendly layout.

#### 4. Images & Design

* Every page contains themed images (food, restaurant ambiance, logo, backgrounds).
* Images are optimized for different devices and screen sizes.

#### 5. Buttons & Forms

* Rounded, pastel-styled buttons with hover effects and smooth transitions.
* Registration and contact forms include placeholders and validation.

#### 6. Media Queries (Responsiveness)

* **CSS media queries** ensure a fully responsive experience.
* Text, buttons, and layout adjust automatically to different screen widths (mobile, tablet, desktop).

#### 7. jQuery & JavaScript Functionality

* Smooth animations for form transitions and popups.
* jQuery functions control form visibility and switching between Sign In / Sign Up.
* JavaScript dynamically updates the header with the user’s name after login.

---

###  Design and Aesthetics

* Main color palette: **light pink, white, and soft pastel tones** inspired by cherry blossoms.
* Rounded shapes, gentle shadows, and minimalistic layout for a modern, elegant feel.
* Fonts: *Segoe UI* and *Arial* — clean and readable for professional web presentation.

---

### Conclusion

The **Oishiro** project represents a complete and practical example of a modern web application.
It covers all major topics of web development:
**HTML, CSS, Bootstrap, JavaScript, jQuery, Buttons, Media Queries, Images, Navigation, Authentication.**

The result is a visually consistent, user-friendly, and fully responsive restaurant website that integrates functionality with Japanese cultural aesthetics.
